# Todo Web Application

A **Todo Web Application** built with **Next.js** and **TypeScript**, following the provided Figma design. This application allows users to manage their todos efficiently with a clean and modern interface.

## Project Overview

This project is a fully functional Todo Web Application where users can:

- Sign up and log in
- Create, update, and delete todos
- Reorder todos using drag-and-drop
- Manage their profile

The application is built using **Next.js**, **TypeScript**, and integrates modern UI components and state management.

## Pages to Develop

The application includes the following pages:

### 1. Signup Page
- Allows new users to create an account.
- Collects information such as first name, last name, email, and password.
- Form validation included.

### 2. Login Page
- Allows existing users to log in.
- Includes authentication and error handling.

### 3. Todo Management Page
The main dashboard for managing todos. Users can:

- **Create Todos**: Add new tasks with relevant details.
- **Update Todos**: Edit existing tasks.
- **Delete Todos**: Remove tasks permanently.
- **Reorder Todos**: Drag-and-drop functionality to reorder tasks easily.

### 4. Profile Page
- View and update user profile information.
- Upload avatar/profile image.
- Update personal information like name, email, contact number, address, and birthday.

## Technologies Used

- **Next.js** – React framework for server-side rendering and static site generation.
- **TypeScript** – Strongly typed JavaScript.
- **React Query** – Data fetching and state management.
- **Framer Motion** – Drag-and-drop functionality for reordering todos.
- **Tailwind CSS** – Styling and UI components.

## Features

- Fully responsive design based on Figma layouts.
- Form validation for signup and profile forms.
- Real-time updates using React Query.
- Drag-and-drop todo reordering using Framer Motion.
- Profile management with avatar upload.
- Clean and maintainable TypeScript codebase.
